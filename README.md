# Py_to_C_Compiler

Compilador PYTHON → C
=============================

COMO USAR:

1. Execute a main com um arquivo Python:

  '''python3 main.py exemplo.py'''

EXEMPLO DE SAÍDA:

  1 | COMMENT_C          → #
  2 | STRING_C           → Calculadora simples em Python
  3 | KEYWORD_C          → // Função:
  4 | NAME_C             → calcular
  ...
